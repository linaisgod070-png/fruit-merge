# 水果合成游戏 - 改进计划

## 上瘾机制（待实现）
1. ⭐ 星级评价系统 - 每局结束给1-3星，让人想拿满星
2. 🎯 每日挑战 - 每天一个目标分数，完成有特殊标记
3. 💥 特殊水果 - 随机出现炸弹果（消除周围）、彩虹果（万能合成）、冰冻果（暂停下落）
4. 🔥 连击加分 - 连续合成倍率递增（x2, x3, x4...）
5. 📈 等级系统 - 累计分数升级，解锁新皮肤/背景
6. 🎁 成就系统 - "第一次合成西瓜"、"单局1000分"等
7. 🎵 背景音乐 - 轻快的循环BGM

## UI 改进
1. 合成时的缩放动画（新水果弹出效果）
2. 分数飘字（+8, +16 从合成位置飘出）
3. 背景随分数变化（越高分背景越华丽）
4. 水果掉落时的拖尾效果
5. 游戏结束时的水果爆炸动画

## 差异化特色
1. 特殊水果是最大差异点 - 原版 Suika 没有
2. 每日挑战 + 排行榜 = 社交驱动
3. 成就系统 = 长期留存
